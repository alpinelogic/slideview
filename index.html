Slideview.js

Why another slider?

Because every content slider I have tried exeeds 1,000 lines of code. If you only need one feature instead of 99 and you want to support only modern Browsers (IE10+), then Slideview may work for you. If non-modern Browsers are a concern, then try Slideview jQuery plugin.

Features

  • Lightweight lib with VanillaJS.
  • Uses CSS transitions for sliding effect.
  • All slides use percent widths, which means that we don't run any JS code when the Browser window resizes (except for Safari when choosing so). The slides automatically get resized by the Browser.
  • Public API to move the slides programmatically.
  • Easily stylable with CSS.
  • 2 separate libs: Vanilla JavaScript & a jQuery plugin.
  • Free as in "free beer".

Warning: Slideview doesn't try to be perfect for every Browser and doesn't try to fix pixel rounding issues in certain Browsers (except for Safari Desktop & only if you choose to use the Browser sniffing helper).

Usage

First you must include the core CSS file and the JS lib.

<link href="css/slideview.css" rel="stylesheet">
<script src="js/slideview.js"></script>

Then create the HTML.

<div class="slideview boxes" id="dark-boxes">
  <div class="inner">
    <div class="offscreen">
      <div class="slide">
        <h3>1>/h3>
      </div>
      <div class="slide">
        <h3>2</h3>
      </div>
      <div class="slide">
        <h3>3</h3>
      </div>
      <div class="slide">
        <h3>4</h3>
      </div>
      <div class="slide">
        <h3>5</h3>
      </div>
    </div>
  </div>

  <nav class="nav">
    <a href="#prev" class="prev">prev</a>
    <a href="#next" class="next">next</a>
  </nav>
</div>

Now you can style Slideview as you please.

/* Here are the styles for the carousel above. */
.boxes .slide { text-align: center; }
.boxes .slide h3 {
  margin: 0 4px;
  padding: 2.25em 1em;
  color: #E6E6E6;
  background: #363A41;
}
.boxes .active h3 { background: #594A69; }
.boxes .nav {
  margin: 1em;
  text-align: center;
}
.boxes .nav a {
  display: inline-block;
  margin: 0 .125em;
  padding: .325em 1em;
  font-size: 18px;
  color: #FCFCFC;
  background: #667292;
  text-decoration: none;
  opacity: .88;
}
.boxes .nav a:hover { opacity: 1; }

Create a new instance in JavaScript.

var slideview = new Slideview('#dark-boxes', {
  slidesToShow: 5
});
